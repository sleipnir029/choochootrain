# PRX Predictor

Valorant match prediction system focused on Paper Rex, with a league-wide data layer.

## Setup

Python 3.11 (conda env `choochoo`):

```
conda activate choochoo && pip install -r requirements.txt
```

`requirements.lock.txt` holds the full pinned dependency tree for reproducible installs.

See `PRX_PREDICTOR_SPEC.md` for design decisions and `docs/` for tasks and progress.
